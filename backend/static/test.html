This page demonstrates the embeddable chat widget. Click the chat button in the bottom-right corner.

Try these questions:
  • What documents do you have?
  • Summarize the main document
  • What are the key procedures?

Embed Options

Option 1: Floating Chat Widget (recommended)

Add this one line to any website — shows a chat bubble in the corner:

<script src="https://your-server:8080/widget.js" data-title="Document Assistant" data-color="#1976d2"></script>

Option 2: Inline iframe

Embed the full chat directly into a page section:

<iframe src="https://your-server:8080/embed" width="100%" height="600" style="border:none;border-radius:12px"></iframe>

Option 3: Public URL

Share this link directly: /embed

Features

All embed options include: streaming answers, inline citations [p5], PDF document viewer, follow-up suggestions, source badges, thumbs up/down feedback, tool call visualization, and copy button.
